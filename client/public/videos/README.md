# Sample Videos

Place your sample video files in this directory with the following names:

- `english-sample.mp4` - English language video
- `spanish-sample.mp4` - Spanish language video
- `french-sample.mp4` - French language video

These videos will be used as sample content for the Lingua language learning platform.

## Video Requirements

- Format: MP4, WebM, or other browser-supported formats
- Recommended: Short videos (1-5 minutes) with clear speech
- The videos should contain spoken content in the target language
